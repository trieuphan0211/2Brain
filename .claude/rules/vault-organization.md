# Rule: Vault Organization

## Purpose

Keep the second brain tidy by enforcing a clear folder structure.

## Rule

1. All new English-learning notes MUST be created inside `D:\2Brain\english\` under the appropriate subfolder.
2. Quick, unprocessed captures may go to `english\00_Inbox\`, but the user should process them within 24 hours.
3. Do NOT create English-learning notes in the vault root, in `Daily_Notes\`, or in `.claude\`.
4. When the user asks to "save," "add," or "create" English content, ask for or infer the correct topic/skill folder.
5. When moving a note from `00_Inbox`, update any internal links and add the required YAML frontmatter.

## Folder Decision Tree

- Schedule / plan / tracker → `english\01_Study_Plan\`
- Vocabulary / word lists / collocations → `english\02_Vocabulary\`
- Grammar rules / exercises → `english\03_Grammar\`
- Listening / Reading / Writing / Speaking strategies → `english\04_Skills\`
- Personal mistakes / corrections → `english\05_Error_Log\`
- Mock test answers / analysis → `english\06_Mock_Tests\`
- Templates → `english\07_Templates\`
