# Matching Headings

## Rule

Match a heading (main idea) to each paragraph or section.

## Strategy

1. Read the list of headings first.
2. Read the first and last sentence of each paragraph.
3. Identify the main idea — not just a detail.
4. Cross off headings as you use them.

## Tips

- The heading should cover the whole paragraph.
- Look for synonyms between the heading and paragraph.
- Don't match just because you see the same word.
