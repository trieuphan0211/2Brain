# Essay Structure

## The 4-Paragraph Structure

1. **Introduction**
   - Paraphrase the question
   - State your position / outline

2. **Body Paragraph 1**
   - Topic sentence (main idea)
   - Explanation
   - Example

3. **Body Paragraph 2**
   - Topic sentence (second main idea)
   - Explanation
   - Example

4. **Conclusion**
   - Restate your position
   - Summarize main ideas

## Introduction Template

> It is often argued that [issue]. In my opinion, I believe that [opinion] because [reason 1] and [reason 2].

## Conclusion Template

> In conclusion, [restate opinion]. This is because [reason 1] and [reason 2].

## Tips for Band 5.0

- Answer the question directly.
- Use simple but correct sentences.
- Use at least 3–4 different linkers.
- Aim for 250–280 words.
