# Task 1 Overview

## Structure

1. **Introduction** — paraphrase the question (1 sentence)
2. **Overview** — main trend / main feature (2 sentences)
3. **Body paragraph 1** — details and data
4. **Body paragraph 2** — details and data

## Key Language

- increase, decrease, rise, fall, drop
- peak, reach a low, remain stable, fluctuate
- the highest, the lowest, the largest, the smallest
- by contrast, compared to, whereas, while

## Tips

- No opinion. Only describe what you see.
- Include specific numbers and dates.
- Overview is the most important paragraph for Band 5.0+.

## Example Overview

> Overall, the number of people using public transport increased significantly between 2000 and 2010, while car usage remained relatively stable.
